### 封装 axios（二次封装）

使用普通的 Axios 时，通常需要在每个请求中重复配置参数，例如请求的 URL、方法、头信息等。

```ts
import axios from 'axios';
axios.get('url')
	 .then((response) => {
		console.log(response);
	 })
	 .catch((error) => {
		console.log(error);
	 });
axios.post('posturl', { key: '1', key2: '2' })
     .then((response) => {
	console.log(response);
});
```

二次封装的 axios：

```ts
首先封装axios的实例为request.js;

import axios from 'axios';
// 创建一个axios对象
const instance = axios.create({
	timeout: 5000,
});
instance.interceptors.request.use(
	(config) => {
		if (config.methods === 'POST' || config.methods === 'PUT') {
			config.headers = {
				'Content-Type': 'application/json', 
                //配置请求头
			};
			//如有需要：注意使用token的时候需要引入cookie方法或者用本地localStorage等方法，推荐js-cookie
			//const token = getCookie('名称');//这里取token之前，你肯定需要先拿到token,存一下
			//if(token){
			//config.params = {'token':token} //如果要求携带在参数中
			//config.headers.token= token; //如果要求携带在请求头中
			// 可以进行序列化，序列化的作用是什么？
			config.data = JSON.stringify(config.data);
		}
		return config;
	},
	(error) => Promise.reject(error);
);

instance.interceptors.response.use(
	(response) => {
		return response.status === 200
			? Promise.resolve(response)
			: Promise.reject(response);
	},
	(error) => {
		if (error && error.response) {
			// 1.公共错误处理
			// 2.根据响应码具体处理
		switch (error.response.status) {
			case 400:
				error.message = '错误请求';
				break;
			case 401:
				error.message = '未授权，请重新登录';
				break;
			case 403:
				error.message = '拒绝访问';
				break;
			case 404:
				error.message = '请求错误,未找到该资源';
				window.location.href = '/NotFound';
				break;
			case 405:
				error.message = '请求方法未允许';
				break;
			case 408:
				error.message = '请求超时';
				break;
			case 500:
				error.message = '服务器端出错';
				break;
			case 501:
				error.message = '网络未实现';
				break;
			case 502:
				error.message = '网络错误';
				break;
			case 503:
				error.message = '服务不可用';
				break;
			case 504:
				error.message = '网络超时';
				break;
			case 505:
				error.message = 'http版本不支持该请求';
				break;
			default:
				error.message = `连接错误${error.response.status}`;
			}
		}
		return Promise.reject(error);
	}
);

export default instance;
```

然后是封装 axios 请求：

```ts
对每个请求有不同的method：
import request from "./axios.js"
const http = {
    get(url, params) {
        config = {
            url: url,
            method: 'GET',
        }
        if (params) {
            config.params = params;
        }
        return request(config);
    },
    post(url, params) {
        config = {
            url: url,
            method: 'POST',
        }
        if (params) {
            config.data = params;
        }
        return request(config);
    },
    put(url, params) {
        const config = {
            method: 'put',
            url: url
        };
        if (params) config.params = params;
        return request(config);
    },
    delete(url, params) {
        const config = {
            method: 'delete',
            url: url
        };
        if (params) config.params = params;
        return request(config);
    }
}
export default http;
```

再就是实现的方式直接调用这个 http 里面的 get 就行了。

```ts
import http from './http';

// 发起 GET 请求
http.get('/data', { id: 1 })
	.then((response) => {
		console.log(response.data);
	})
	.catch((error) => {
		console.error('请求失败:', error);
	});

// 发起 POST 请求
http.post('/data', { name: 'John', age: 30 })
	.then((response) => {
		console.log(response.data);
	})
	.catch((error) => {
		console.error('请求失败:', error);
	});
```
